<template>

The template element is used to declare fragments of HTML that can be cloned and inserted in the document by script.


Try It Out



The Template:

    <template id="templateToClone">
        Hi <span class="userName"></span>!
        <img src="glad.png" width="42" height="42">
    </template>

The JavaScript:

importNode 
document.importNode(externalNode, [deep])

Creates a copy of a node from an external document that can be inserted into the current document.

arguments
externalNode
The node from another document to be imported. 
deep
An optional boolean, indicating whether the descendants of the imported node need to be imported. default=false

Docs: https://developer.mozilla.org/en-US/docs/Web/API/Document/importNode

Spec: https://html.spec.whatwg.org/multipage/scripting.html#the-template-element
